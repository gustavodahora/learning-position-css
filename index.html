How to understand position in CSS

In this example, we will use three boxes. (Or more =D)

Two simple things to understand are:

position: absolute;

position: relative;

To position objects using only position in CSS, you need to understand the Family concept.


The concept is, TAG father always need to have position relative, because absolute father will find the next father relative, if you don't put this attribute, your block will go to top of the page. (TAG html will be the relative father.)

How position the absolute elements?

You can have many ways to position using "position: absolute;", but the way I use it is through basic CSS positioning:

right: 0;

left: 0;

top: 0;

bottom: 0;

The value of the element in the browser in standart is: 0 for all the elements.

You can set across all units of measurement of CSS.
